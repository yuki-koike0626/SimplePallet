# SimplePallet

macOS 向けのシンプルなウィンドウ管理アプリ。キーボードショートカットで最前面ウィンドウを「最大化 / 2分割 / 3分割」に即座にスナップできます。

## 📖 使い方

### 1. インストール
1. ダウンロードしたDMGファイルを開きます。
2. アプリコンを「Applications」フォルダにドラッグ＆ドロップします。
3. アプリを起動し、アクセシビリティ権限を許可してください。

### 2. キーボードショートカット
インストール後、すぐに以下のショートカットが使えます：

**基本操作**
- **最大化**: `⌘ + ↑`
- **左半分**: `⌘ + →`
- **右半分**: `⌘ + ←`

**3分割配置**
- **左1/3**: `⌥ + ⌘ + ←`
- **中央1/3**: `⌥ + ⌘ + ↑`
- **右1/3**: `⌥ + ⌘ + →`

※ ショートカットはメニューバーから変更可能です。

### 3. 自動アップデート
新しいバージョンが公開されると自動で通知され、ワンクリックでアップデートできます。
